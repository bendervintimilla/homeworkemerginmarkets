# Plan de Acción: Presentación "The Silent Shift" (Algorithmic Orchestration)

Este plan transforma tu presentación sobre **"The Silent Shift: From Local Execution to Algorithmic Orchestration"** en una experiencia web narrativa e interactiva. El objetivo es visualizar la tensión entre la ejecución humana local y la orquestación algorítmica.

## 1. Stack Tecnológico
- **Framework**: **Vite + React** (Rápido y robusto).
- **Estilos**: **Tailwind CSS** (Diseño limpio y profesional).
- **Animaciones**: **Framer Motion** (Para explicar gráficos complejos paso a paso).
- **Iconos**: **Lucide React** (Iconografía industrial/técnica).

## 2. Estética y Diseño (Industrial Tech - NO BANANAS)
- **Concepto Visual**: "Modern Industrial Cyberpunk meets McKinsey". Limpio, profesional, pero con toques futuristas.
- **Paleta de Colores**: 
  - *Base*: Gris Pizarra (Slate 900) y Blanco Técnico.
  - *Acento*: Naranja Seguridad (Safety Orange) para alertas/humanos.
  - *Digital*: Azul Cian (Electric Blue) para el "sistema".
- **Tipografía**: Una combinación de **Inter** (UI) y **Roboto Mono** (Datos/Código) para enfatizar la naturaleza técnica.

## 3. Estructura del Sitio (Scrollytelling)

### Sección 1: Hero - "The Silent Shift"
- **Visual**: Animación de transición entre una mano humana y un brazo robótico (metáfora del relevo/baton pass).
- **Título**: The Silent Shift.
- **Subtítulo**: From Local Execution to Algorithmic Orchestration.
- **Interacción**: Scroll para iniciar la "transformación".

### Sección 2: La Definición (The Definition)
- **Concepto**: Comparación lado a lado.
- **Izquierda (Antiguo)**: Task Allocation (Social).
- **Derecha (Nuevo)**: Coordination (System-Mediated).
- **Visual**: Diagramas de red que se reorganizan de "caóticos/sociales" a "estructurados/algorítmicos".

### Sección 3: Rewiring the Factory Nervous System
- **Visual**: Efecto de pantalla dividida (Split Screen).
  - *Local Trust*: Foto industrial con filtro cálido/grano.
  - *System Trust*: Dashboard digital limpio y frío.
- **Contenido Interactiva**: Tabla comparativa que cambia al hacer hover (Authority, Decision Speed, Recovery).

### Sección 4: The Green Dashboard Paradox
- **Visual**: Gráfico interactivo animado ("The Timeline").
- **Narrativa**: El usuario hace scroll y la línea de tiempo avanza:
  1.  **Trigger** (Sensor drift).
  2.  **Conflict** (Human eye vs Dashboard).
  3.  **Delay** (Hesitation).
  4.  **Impact** (Downtime).
- **Insight**: Resaltar "Blind Trust" vs "Verified Reality".

### Sección 5: Governing the Hybrid Line (Solución)
- **Visual**: 3 Pilares 3D (Tiered Stop, Hybrid Ground-Truth, Empowered Override).
- **CTA Final**: "Restore the Human-in-the-Loop".
